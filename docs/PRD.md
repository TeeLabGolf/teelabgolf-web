# TeeLab AI — 产品需求文档 (PRD)

## 产品概述

TeeLab AI 是一款面向海外高尔夫球友的共享日历应用，帮助球友在私密小组内发布 Tee Time、报名组队、实时记分和统计成绩。

## 目标用户

海外高尔夫爱好者，经常在不同朋友圈组织球局，需要一个轻量工具来协调 Tee Time、记分和追踪表现。

## 平台策略

- **Web App 先行**（快速验证产品，天然适配链接分享）
- **原生 App 紧随**（更顺畅的体验）

## 功能模块

### 1. 群组系统

#### 1.1 群组基本信息

- 私密小组，每个群有一个**短数字编号**（唯一 ID）
- 群名称、群公告
- 数字编号可搜索、可分享

#### 1.2 角色与权限

| 操作 | 群主 | 管理员 | 普通成员 |
|------|:----:|:------:|:--------:|
| 审批加入申请 | ✓ | ✓ | ✗ |
| 移除成员 | ✓ | ✓ | ✗ |
| 群公告管理 | ✓ | ✓ | ✗ |
| 指定/撤销管理员 | ✓ | ✗ | ✗ |
| 转让群主 | ✓ | ✗ | ✗ |
| 发布 Tee Time | ✓ | ✓ | ✓ |
| 分享邀请链接 | ✓ | ✓ | ✓ |
| 报名 / 围观 | ✓ | ✓ | ✓ |

#### 1.3 入群流程

1. 获取入口：通过**分享链接**或**搜索数字编号**
2. 非成员点击后提交**加入申请**
3. 群主或管理员**审批**
4. 审批通过后成为群成员

#### 1.4 群主规则

- 群主可指定**多个管理员**
- 群主退出前**必须先转让**群主身份

#### 1.5 订阅限制

- 免费用户限制可加入的群数量
- Pro 订阅解锁更多群名额

---

### 2. 球场管理

#### 2.1 公共球场库

- 全局共享，所有群可搜索使用
- 球场信息：名称、地址、洞数（9/18 洞）、每洞标准杆、多套 Tee 标签

#### 2.2 球场数据录入

- 创建球局时发现球场不存在，用户可**自行创建**
- 录入每洞标准杆时，**默认全部为 Par 4**，用户只需调整 Par 3 和 Par 5 的洞
- 新建球场状态为**待审核**（仅创建者可用）
- **平台管理员**审核通过后，球场全局公开

#### 2.3 多套 Tee

- 每个球场可维护多套 Tee（蓝 / 白 / 红 / 金 等）
- Tee 仅作为**标签**，各套 Tee 共享同一套洞标准杆
- 创建球局时选择球场 + 选择 Tee
- 记分板显示本场所打 Tee

#### 2.4 球场洞数支持

- 支持 **18 洞**和 **9 洞**球场
- 9 洞球场同样支持完整的标准杆配置

---

### 3. Tee Time 管理

#### 3.1 发布球局

- 群内**任意成员**均可发布
- 必填信息：日期时间、球场、人数上限
- 可选信息：费用（仅展示）、备注
- Tee 选择和开球洞号**不在创建时设置**，而是在**开始记分时**确定（Tee 因人而异，开球洞号由球场 Starter 现场决定）

#### 3.2 报名与候补

- 成员报名，未满员时直接加入
- **满员后**进入候补队列（按报名时间 FIFO 排序）
- 已报名球员**退出**后，候补队列中排在最前的球员**自动递补**

#### 3.3 费用

- 仅做**展示**，不涉及 App 内支付或分摊

#### 3.4 球局展示

- **日历视图**：按月/周查看球局分布，有球局的日期有标记
- **列表视图**：按时间排序，支持状态筛选（即将开始、进行中、已结束）
- **"我的球局"**：快速筛出当前用户已报名 / 已参与的球局

---

### 4. 记分系统

#### 4.1 开始记分

- 开始记分时设置：**开球洞号**（Hole 1 或 Hole 10，由球场 Starter 现场决定）、**每位球员的 Tee**（因人而异）
- 基于球场每洞标准杆进行**逐洞记分**
- 记分板按**实际打球顺序**排列（支持从后九洞开始），统计时映射回正确洞号

#### 4.2 多人协作记分

- 允许**多人同时记分**
- 可以**替他人记分**

#### 4.3 支持赛制

- **比杆赛**（Stroke Play）：记总杆数
- **比洞赛**（Match Play）：逐洞比较输赢
- **2v2**（Four-ball / Foursome）：团队对抗

#### 4.4 实时围观

- 组内成员可**实时查看**进行中比赛的记分板
- 实时更新，类似 Live Leaderboard

#### 4.5 球局聊天

- 每个球局有独立的**聊天区**，用于赛前协调、赛中讨论、赛后复盘
- 每个洞支持独立的**评论/讨论**，可针对某一洞的精彩瞬间展开聊天

---

### 5. 数据统计

#### 5.1 统计维度（组内）

所有统计数据基于成员**在本组内**的比赛记录：

| 统计项 | 说明 |
|--------|------|
| 成绩记录 | 每场比赛的完整记分卡 |
| Handicap | 差点计算与变化趋势 |
| 平均差点 | 近期比赛的平均差点 |
| 出场频率 | 参与球局的次数和频率 |
| 成绩曲线 | 每场成绩的趋势图 |

#### 6.1 登录方式

- Apple 登录
- Google 登录

#### 6.2 订阅模式

- **免费版**：限制可加入的群数量，基础功能

### 7. 站内通知

#### 7.1 消息中心

- 站内通知列表，支持未读角标
- 点击通知跳转到对应页面

#### 7.2 通知场景

| 场景 | 接收人 |
|------|--------|
| 有人申请加入群 | 群主 / 管理员 |
| 加入申请被批准 / 拒绝 | 申请人 |
| 新球局发布 | 群内所有成员 |
| 有人报名 / 退出你发布的球局 | 球局发布者 |
| 候补递补成功 | 被递补的球员 |
| 球局即将开始（提前提醒） | 已报名球员 |
| 有人更新了记分 | 同场球员 |

#### 7.3 Push 推送（原生 App 核心能力）

原生 App 的核心价值之一是 **Push Notification**。球局相关通知天然时效性强，必须及时触达用户。

**推送优先级分级：**

| 优先级 | 场景 | 说明 |
|--------|------|------|
| 🔴 高 | 候补递补成功 | 用户需要立即确认是否参加 |
| 🔴 高 | 球局即将开始（提前 1 小时） | 避免迟到 |
| 🟡 中 | 新球局发布 | 热门时段名额有限，先到先得 |
| 🟡 中 | 有人退出你报名的球局 | 可能影响是否继续参加 |
| 🟢 低 | 有人报名你发布的球局 | 了解即可 |
| 🟢 低 | 有人更新了记分 | 围观用户关注 |
| 🟢 低 | 入群申请 / 审批结果 | 不紧急 |

**通知渠道策略：**

| 渠道 | 适用场景 |
|------|----------|
| Push（App） | 所有通知，App 端主力渠道 |
| 站内通知 | 所有通知，Web + App 通用 |
| Email | 仅重要摘要（如每周球局汇总），V2 |

**通知偏好设置：**
- 用户可按场景开关 Push 通知
- 可设置免打扰时段

#### 7.4 后续扩展（V2）

- Email 通知（每周球局摘要）
- 通知偏好细粒度设置

## 设计规范

### 平台策略

- 目标平台：**Web 移动端**（先行）+ **原生 App**（紧随）
- 不需要适配 Web 桌面端
- 两端共享同一套视觉设计语言（配色、字体、间距、圆角、阴影体系）
- **原生 App 的核心价值**：Push Notification（球局通知时效性强，必须及时触达）、原生手势体验

### 平台差异

| 维度 | Web 移动端 | 原生 App |
|------|-----------|----------|
| 导航 | Header 和功能入口**不固定**，随页面滚动（避开浏览器地址栏和工具栏对可视区域的挤占） | 可以使用**固定导航栏和 Tab Bar**（无浏览器 UI 干扰） |
| 弹窗 | 底部 Sheet 上滑 | 底部 Sheet 上滑（一致） |
| 操作入口 | 内联按钮（如 "+ New Tee Time"） | 可使用 FAB 浮动按钮 |
| 抽屉菜单 | 右侧抽屉 | 右侧抽屉（一致） |
| 手势 | 有限（浏览器限制） | 可使用原生手势（滑动返回、下拉刷新等） |

### 视觉风格

- **定位**：Premium Sporty-Luxe，高端运动感
- **字体**：Bricolage Grotesque（标题/数字）+ Plus Jakarta Sans（正文），App 端可替换为系统等效字体
- **配色**：以深绿为主色调（高尔夫主题），暖色中性灰，避免冷蓝调
- **卡片**：轻微毛玻璃效果（`backdrop-filter: blur`），细边框 + 柔和分层阴影
- **交互**：弹性动效（`cubic-bezier(0.34, 1.56, 0.64, 1)`），按钮有深度感（内阴影高光）
- **原型参考**：`prototype/index.html`

## MVP 范围

### MVP（V1）

- 群组系统（创建、加入、权限管理）
- 球场公共库（创建、审核、多套 Tee 标签）
- Tee Time 发布、报名、候补
- 日历 / 列表 / 我的球局视图
- 逐洞记分（比杆赛）
- 实时围观记分板
- 基础统计（成绩记录、成绩曲线）
- 站内通知（消息中心 + 未读角标）

### V2

- 比洞赛 / 2v2 赛制
- 好友管理（跨群社交关系）
- Handicap 与差点统计
- Pro 订阅与付费
- 原生 App
- 围观互动（点赞、评论）
